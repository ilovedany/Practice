
Это простое приложение демонстрирует использование HttpClient для получения данных JSON с сервера и их вывода в консоль.


При запуске вы увидите список пользователей, полученных с сервера.

## Структура проекта

- MainProgram.cs: Основной файл программы, содержащий метод HttpMethodAsync, который выполняет HTTP-запрос к серверу и выводит данные в консоль.
- Класс модели User, представляющий структуру данных, получаемых от сервера.
